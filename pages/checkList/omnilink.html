• Iniciar o checklist no Krona One para sinalizar que já está sendo acompanhado
  • Entrar no Omnilink geral para localizar se o veículo tem cadastro
  • Caso localize vincule o veículo no seu usuário
  • Caso não tenha cadastro solicite a ficha de ativação para a transportador via e-mail
  • Cadastre a ficha de ativação com o transportador e frota respectivo a transportador
  • Após cadastrar ou vincular o veículo no seu usuário verifique se está espelhado
  • Se não tiver espelhado peça os dados para a transportador para fazer
  • Após espelhar o veículo verifique se está posicionando na antena celular
  • Confira o cadastro do veículo se está conforme com o tipo de caminhão cadastrado na solicitação de checklist no krona one
  • Envie os comandos todos com perfil krona
  • Verifica se todas as configurações foram recebidas pelo veículo em (histórico-telecomandos)
  • Transmita as mensagens padrão krona predefinidas
  • Entre em contato com o condutor
  • Peça para ele entra na cabine, fechar as portas e virar a chave
  • Peça para manter o veículo engatado caso tenha o sensor
  • Peça para o condutor ativar o modo interativo com a senha de teclado dele
  • Após ativar peça para ele apertar (menu-saída-colocar a sua senha) para que o sistema fique em transito
  • Quando ficar em trânsito pede para ele apertar o botão de pânico por 5 segundos e soltar e abrir a porta do motorista
  • Após gerar, peça para que ele feche a porta do motorista
  • Envie o comando de desbloqueio
  • Peça para o condutor abrir a porta do carona e descer do veículo
  • Quando gerar peça para que feche a porta do carona
  • Envie o comando de desbloqueio
  • Peça para que solte a tomada do desengate
  • Quando gerar peça para engatar
  • Se não tiver sensor de baú envie o comando para desativar o veículo
  • Aprove o teste no sistema kronaone
  • Coloque na observação “TESTE APROVADO 00/00”
  • Caso ele tenha baú, desative o veículo e peça para que ele encoste a porta do baú ou coloque algo no buraco da trava para impedir que ela desça
  • Após isso pode ativar o sistema e quando ficar em local autorizado pede para o condutor abrir o baú
  • E para gerar “arrombamento de baú”
  • Após gerar pode desativar o veículo e aprovar o teste no kronaone
  • Coloque na observação “TESTE APROVADO 00/00”
