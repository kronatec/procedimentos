Checklist Autotrac

  • Iniciar o checklist no Krona One para sinalizar que já está sendo acompanhado
  • Identificar o MCT no Krona One e buscar no autotrac para ver se esta espelhado ou não
  • Caso não esteja espelhado peça os dados ou o espelhamento para a transportador
  • Caso esteja espelhado vincule o veículo no conjunto de controle que você utiliza
  • Verifique se o OBC está ativo
  • Caso não esteja, envie o comando “solicitar firmware”
  • Após ele gerar no sistema vá até a configuração de OBC e coloque o perfil correto
  • Envie o comando “limpar funções logicas” e “geral conf. básica”
  • Entre em contato com o condutor e informe que que iremos iniciar o teste
  • Peça para ele ficar no veículo com tudo fechado e engatado
  • Envie o comando de “ativa checklist” e “inicio de viagem n3”
  • Peça para ele abrir uma das portas e soltar o cabo do desengate
  • Quando ele violar envie o comando “solicitar tabela de status” e espere chegar a tabela com as violações
  • Após chegar peça para que ele feche a porta que abriu e engate a carreta
  • Quando ele fechar envie o comando “solicitar tabela de status” e espere ela retornar vazia
  • Quando vim vazia peça para ele abrir a outra porta e caso tenha baú, a porta do baú também
  • Quando ele violar envie o comando “solicitar tabela de status” e espere chegar a tabela com as violações
  • Quando vim a tabela com as violações pede para ele fechar as portas “cabine e baú”
  • Após fechar, envie o comando “solicitar tabela de status”
  • Quando a tabela vim vazia pode avisar o condutor que o teste foi aprovado
  • Envie os comandos de “desativa checklist, liberar veículo e desligar sirene, autorizar carona e desengate”
  • Aprove o checklist no kronaone
  • Coloque na observação “TESTE APROVADO 00/00”
  • Para testar o BOTÃO DE PÂNICO, enviar o comando obrigatório “teste de botão de pânico”, o condutor poderá acionar o botão por uma hora sem custo. Sempre enviar o comando de teste do botão de pânico antes de pedir para o condutor apertar o pânico
