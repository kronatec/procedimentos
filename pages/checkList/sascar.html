Checklist Sascar

  • Iniciar o checklist no Krona One para sinalizar que já está sendo acompanhado
  • Verificar se o veículo está espelhado
  • Caso não esteja espelhado peça os dados de espelhamento para a transportadora
  • Localizar o veículo na tecnologia e incluir no seu grupo
  • Verificar os sensores e atuadores se estão conforme com o cadastro do veículo na solicitação de checklist no krona one
  • Verificar se o veículo está posicionando em “GSM”
  • Embarcar os macros caso não tenha ou esteja incorreta
  • Iniciar o checklist no krona one
  • Entrar em contato com o condutor
  • Pedir a ele que entre na cabine
  • Deixar tudo fechado e engatado
  • Vire a ignição, não precisa ligar o motor
  • Envie os comandos (ativar todas as saídas, ativar todas as entradas, início de operação e desinibir todos os sensores)
  • Peça para o condutor enviar a macro de início de viagem
  • Após chegar a macro, peça para que ele aperte o botão de pânico durante 5 segundos e soltar
  • Após gerar o botão de pânico, peça para que ele abra a porta do motorista
  • Após gerar, peça para que feche a porta do motorista
  • Envie o comando de “reset alarme mantendo perfil de segurança”
  • Após sair de alarme peça para que abra a porta do carona
  • Após gerar, peça para que desça do veículo e feche a porta do carona
  • Envie o comando de “reset alarme mantendo perfil de segurança”
  • Caso tenha sensor do baú, trave o mesmo e peça para ele tentar abrir
  • Se ele abrir mesmo travado termine o teste mas reprove pela trava
  • Caso ele não consiga abrir destrave e peça para o mesmo abrir o baú
  • Quando gerar aberto pede para ele fechar
  • Gerando fechado envie o comando “reset alarme mantendo perfil de segurança”
  • Após desbloquear e sair de alarme peça para que solte o desengate (caso tenha)
  • Após gerar peça para que engate o cabo
  • Gerando tudo fechado pode aprovar o teste no krona one
  • Pedir ao condutor que mande os macros até o fim de viagem
  • Enviar os comandos (desativar todas as saídas, desativar todas as entradas, inibir todos os sensores e fim de operação)
  • Coloque na observação do veículo “TESTE APROVADO 00/00”
